クローンする
    git clone リポジトリパス

リポジトリを新規作成し登録する
    github.comでリポジトリを作成

    ルートフォルダ上でgit init
    git init

    ステージング
        個々のファイルをステージングする
        git add hello.html
        変更のあったファイルをすべてステージングする
        git add -A

    コミットする
    git commit -m "メッセージ"

    リモートリポジトリとローカルリポジトリをリンクさせる
    git remote add アクセス先 gitのurl
    git remote add origin https://github.com//awesome.git

    pushする
    git push アクセス先 ブランチ名
    git push origin master

stash
    stash登録
        git stash save "メッセージ"

    最新のstashを反映
        git stash pop

ある時点のcommitに戻す
    git reset --hard ハッシュ値

ファイルの変更を取り消す
    ```
    git chekout -- <ファイル名>
    git chekout -- <ディレクトリ名>
    ```

    すべての変更を取り消す
    ```
    git chekout -- .
    ```

remoteリポジトリを変更する
    1. 現状のremoteリポジトリを確認
    git remote -v
    2. 新しいものに設定
    git remote set-url origin コピーしたurl

ブランチ取得
    git fetch origin ブランチ名

    全部ブランチをリモートからプル
        ```
        for remote in `git branch -r`; do git branch --track ${remote#origin/} $remote; done
        git fetch --all
        git pull --all
        ```
